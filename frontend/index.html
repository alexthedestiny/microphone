Welcome to Nextlevel.Ai cross-browser Speech To Text service v 1.0!

How it works:

Available as SaaS paid by the hour for voice->text translation, brands can now make it possible for their users to interact with web pages through voice (for ex entering search queries, or chatting via web chat). Recording a voice for translation is very easy - just press and hold the button to capture your voice and release when finished.

Recognized voice will appear in the demo web chat text area.

Current release supports all modern browsers:

Desktop:
-Google Chrome
-Mozilla Firefox
-Opera
-Edge
-Safari
-Internet Explorer

Mobile Android:
-Google Chrome
-Mozilla Firefox
-Opera

Mobile iOS:
-Safari

Next release features:
- Real time streaming recognition
- Start speech recognition by a key word (similar to amazon alexa)
- automatic stop voice recording when silent
